# Prints do dashboard

Salve aqui as imagens usadas no portfolio.

Sugestao:

```text
dashboard-visao-geral.png
dashboard-clientes.png
dashboard-produtos.png
dashboard-qualidade-comercial.png
```

Como gerar:

1. Abra o Metabase local em `http://localhost:3000`.
2. Abra o dashboard.
3. Tire prints das secoes principais.
4. Salve as imagens nesta pasta.

Esses prints sao a forma mais simples e confiavel de apresentar o dashboard em README, GitHub, LinkedIn ou site pessoal.
